
#  错误码

##  错误码说明

返回结果中如果存在 Error 字段，则表示调用 API 接口失败。 Error 中的 Code 字段表示错误码，所有业务都可能出现的错误码为公共错误码，下表列出了公共错误码。
错误码列表

- HTTP返回码

| 错误码 |          错误描述          |
| :----: | :------------------------: |
|  400   |  请求不合法，包体格式错误  |
|  403   |          禁止访问          |
|  404   |          请求失败          |
|  500   |       服务器内部错误       |
|  501   |         服务未实现         |
|  502   |    网关错误，服务不可用    |
|  503   |         服务不可用         |
|  504   | 后端服务不可用或者处理超时 |
